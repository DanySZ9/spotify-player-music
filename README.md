# Player Music with API Spotify

A small widget created with Electron that lets you see in real time the song that is playing in your Spotify app.

En la aplicacion el usuario puede:
- Reproducir y pausar la cancion.
- Adelantar la cancion mediante la barra de reproduccion de la cancion.
- Adelantar a la siguiente cancion.
- Retroceder a la cancion anterior.


## CARACTERISTICAS ##
- Login con Spotify OAuth.
- Reproduccion de la cancion actual.
- Control de reproduccion.
- Widget minimalista.
- Renovacion automatica de token.

## TECNOLOGIAS UTILIZADAS ##
- HTML / CSS / JavaScript
- Electron
- Node.js
- Spotify Web API

## INSTALACION ##
1. Clonar el repositorio.
2. Instalar las dependencias con el siguiente comando:
   
npm install

4. Iniciar la aplicacion con cualquiera de los siguiente comando:

npm start
npm run dev
